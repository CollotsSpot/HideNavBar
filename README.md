# Fullscreen Gestures for Android 10

This is a simple Magisk module that enables Fullscreen Gestures on Android 10.

This module will hide both the navbar and the pill for a Fullscreen Gesture experience


## Requirements
- Android 10 only
- Magisk 18+

## Installation
1. Flash this module.
2. Reboot.
3. Enjoy!
